## Состав репозитория

| Файл / папка | Кейс-задача |
|---|---|
| `case1_array.py` | № 1 — сумма отрицательных элементов массива между max и min |
| `case2_inheritance.py` | № 2 — демонстрация методов базового и производного классов |
| `tourism_db.sql` | № 3 — база данных «Туризм» (MySQL 8.0): 4 справочника + таблица заказов |
| `webapp/` | № 4 — WEB-приложение «Каталог туров» (Flask + MySQL) |

## Запуск

```bash
# кейсы 1-2
python3 case1_array.py
python3 case2_inheritance.py

# кейс 3 — создание БД
mysql -u root -p < tourism_db.sql

# кейс 4 — веб-приложение
pip install flask flask_sqlalchemy pymysql
export DATABASE_URI="mysql+pymysql://user:pass@localhost/tourism"
python3 webapp/app.py
```
